# BookieCookie

A mobile book tracking application that allows users to manage personal libraries, track reading progress, rate books, and save reading notes and quotes.

## Features

### Book Management
- Add, update, search, and delete books
- Organize personal reading library
- Track reading status and ratings

### Smart Book Input
- Search books using Google Books API
- Add books by barcode scanning
- Upload book covers

### Reading Notes & Quotes
- Write personal notes while reading
- Add quotes manually
- Extract quotes using OCR text recognition

### Reading Tracker
- Stopwatch/timer while reading
- Track total reading time
- Reading activity history

### Reports & Statistics
- Reading streak tracking
- Weekly/monthly reading analytics
- Reading heatmap visualization
- Goal tracking dashboard

### Achievement Sharing
- Unlock reading achievements
- Export aesthetic achievement badges
- Share accomplishments on social media
